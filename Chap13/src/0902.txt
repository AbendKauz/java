		- Vector
			- 저장할 객체 타입을 타입 파라미터로 표기하고 기본 생성자를 호출하여 생성
		[기본형]
		List<E> list = new Vector<E>();
		List<E> list = new Vector<>();
		// vector의 E타입 파라미터를 생략하면 왼쪽 list에 지정된 타입을 따라감
		
			* 동기화된 메소드로 구성되어 멀티 스레드가 동시에 Vector의 메소드를 실행할 수 없고,
			하나의 스레드가 메소드 실행을 완료해야만 다른 스레드가 메소드 실행을 할 수 있음
			* 멀티 스레드 환경에서 안전하게 객체를 추가 및 삭제할 수 있음
