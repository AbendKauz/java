- 열거타입
	- 한정된 값(열거 상수)을 갖는 타입

	-- 열거타입 생성
		- project -> src -> package (select) -> new -> Enum click
		- 열거타입 이름은 첫글자는 대문자

        열거타입 이름
             ↑
            ----
public enum Week {
       ----
         └───────────→ 열거형(서로 연관된 상수들의 집합)

	MONDAY,	────┐
	TUESDAY,	|
	WEDNESDAY,	|
	THURSDAY,	├──→ 열거 상수
	FRIDAY,		|
	SATURDAY,	|
	SUNDAY	────┘
}	
	- 열거상수는 전부 대문자로 작성할 것
	- 다른 단어가 연결되는 경우 _(under bar)로 표시
	
	* 열거타입의 장점
		- IDE(통합개발환경)의 지원을 받을 수 있다.(자동완성, 오타검증 등)
		- 가독성이 향상되어 소스코드 이해가 높아짐
		- 관련있는 값들을 하나의 변수명으로 묶음 -> 한곳에서 한번에 구성 파악 가능
		- 허용 가능한 값들을 제한 가능(실수로 다른 값이 들어가는 것을 방지함 => 안정성)
		- 내용을 추가해도 Enum코드만 수정하면 됨

		-- 통합개발환경(Integrated Development Environment, IDE)이란
			- 코딩, 디버그, 컴파일, 배포 등 프로그램 개발에 모든 작업을 하나의
				프로그램 안에서 처리하는 환경을 제공하는 소프트웨어
